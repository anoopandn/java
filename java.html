Introduction

Java "Hello! World" Program

class HelloWorld { public static void main(String[] args) { System.out.println("Hello! World"); } }

Output
Hello! World

Explation

  1. class HelloWorld { ... }

    In Java, every application begins with a class definition. In this program, HelloWorld is the name of the class, and the class definition is:

    class HelloWorld {
    ... .. ...
    }

    Every Java application has a class definition,and the name of the class should match the file name in Java.

  2. public static void main(String[] args) { ... }

    This is the main method. Every application in Java must contain the main method. The Java compiler starts executing the code from the main method.

    The main function is the entry point of Java application, and it's mandatory in a Java program. The signature of the main method in Java is:

    public static void main(String[] args) {
    ... .. ...
    }


  3. System.out.println("Hello! World");

    The code above a print statement. It prints the text Hello! World to standard output (your screen). The text inside the quotation marks is called Strin in Java.

Note:

  • Every valid Java Application must hava a class definition (that matches the filename).
  • The main method must be inside the class defintion.
  • The compiler executes the codes starting from the main function.

This is a valid Java program that does nothing.

Java Variables and Literals

Java Variables

In programing, a variable is a container (storage area) to hold data.

To indicate the storage area, each variables should be given a unique name(identifier). Variable names are just the symbolic representation of a memory location. For example
int score = 85;

Here, score is a variable of int data type and we have assigned value 85 to it.

The int data type suggests that the variable can only hold integers.

In the example, we have assigned value to the variable during declaration. However, it's not mandatory.

You can declare variables and assign variables separately. For example

int score;
score = 85;

Java is statically-typed language. It means that all variables must be declared before they can be used.

Rules for Naming Variables in Java

  • Variables must start with either a letter or an underscore, _ or a dollar, $ sign.
  • Variable names cannot start with numbers
  • Variable names can't use whitespace.

There are 4 types of variables in Java programming language:

  1. Instance Variables (Non-Static Fields)
  2. Class Variables (Static Fields)
  3. Local Variables
  4. Parameters

Java Literals

Literals are data used for representing fixed values. They can be used directly in the code. For example

int a = 1;
float b = 2.5;
char c = 'F';

Here, 1, 2.5, 'F' are literals

Here are different types of literals in Java

Boolean Literals

In Java, boolean literals are used to initialize boolean data types. They can store two values: ture and false.

Integer Literals

An integer literal is a numeric value (associated with numbers) without any fractional or exponential part. There are 4 types of integer literals in Java:

  1. binary (base 2)
  2. decimal (base 10)
  3. octal (base 8)
  4. hexadecimal (base 10)

Integer literals are used to initialize variables of integer types like byte, short, int and long

Floating-point Literals

A floating-point literal is a numeric literal that has either a fractional form or an exponential form.

The floating-point literal are used to initialize float and double type variables.

Character Literals

Character literals are unicode character enclosed inside single quotes.

String Literals

A string literal is a sequence of characters enclosed double-quotes.




Java Data Types (Primitive)

Java Data Types

Data types specify the type of data that can be stored inside variables in Java.

Java is a statically-typed language. This means that all variables must be declared before they can be used.

int speed;
Here, speed is a variable, and the data type of variable is int.

The int data type determines that the speed variable can only contain integers.

there are 8 data types predefined in Java programming language, known as primitive data types.

Primitive Data Types

1. boolean type

  • The boolean data type has two possible values, either true or false.
  • Default value: false.
  • They are usually used for ture/false conditions.

Example : Java boolean data type

class HelloWorld { public static void main(String[] args) { boolean flag = true; System.out.println(flag); } }

2. byte type

  • The byte data type can have values from -128 to 127 (8-bit signed two's complement integer).
  • If it's certain that the value of a variable will be within -128 to 127, then it is used instead of int to save memory.
  • default value: 0

Example : Java byte data type

class HelloWorld { public static void main(String[] args) { byte score; score = 84; System.out.println(score); } }

3. short type

  • The short data type in Java can have values from -32768 to 32767 (16-bit signed two's complement integer).
  • If it's certain that the value of a variable will be within -32768 and 32767, then it is used instead of other integer data types (int,long).
  • Default values: 0

Example: Java short data type

class HelloWorld { public static void main(String[] args) { short temperature; temperature = -200; System.out.println(temperature); } }

4. int type

  • The int data type can have values from -2'31 to 2'31-1 (32-bit signed two's complement integer).
  • If you are using Java 8 or later, you can use an unsigned 32- bit integer. This will have a minimum value of 0 and a maximum value 2'32-1.
  • Default value: 0

Example : Java int data type

class HelloWorld { public static void main(String[] args) { int range; range = -6780005; System.out.println(range); } }

5. long type

  • The long data type can have values from -2'63 to 2'63-1 (64-bit signed two's complement integer).
  • If you are using Java 8 or later, you can use an unsigned 64-bit integer with a minimum value of 0 and a maximum value of 2'64-1.
  • Default value: 0

Example : Java long data type

class HelloWorld { public static void main(String[] args) { long range; range = -67800050000L; System.out.println(range); } }

The use of L at the end of -67800050000 , This represents that it's an integral of the long type.

6. double type

  • The double data type is a double-precision 64-bit floating-point.
  • It should never be used for precise values such as currency.
  • Default value: 0.0 (0.0d)

Example : Java double data type

class HelloWorld { public static void main(String[] args) { double number; number = -42.3; System.out.println(number); } }

7. float type

  • The float data type is a single-precision 32-bit floating-point.
  • It should never be used for precise values such as currency.
  • Default value: 0.0 (0.0f)

Example : Java float data type

class HelloWorld { public static void main(String[] args) { float number; number = -42.3f; System.out.println(number); } }

To tell the complier to treat -42.3 as float rather than double, you need to use f or F.

8. char type

  • It's a 16-bit Unicode character.
  • The minimum value of the char data type is '\u0000' (0) and the maximum value of the is '\uffff'
  • Default value: '\u0000'

Example: Java char data type

class HelloWorld { public static void main(String[] args) { char letter; letter = '\u0051'; System.out.println(letter); } }

Here, the Unicode value of Q is \u0051. Hence, we get Q as the output.

Here is another example:

class HelloWorld { public static void main(String[] args) { char letter1 = '9'; System.out.println(letter1); char letter2 = 65; System.out.println(letter2); } }

here, we have assigned 9 as a character (specified by single quotes) to the letter1 variable. However, the letter2 variable is assigned 65 as an integer number (no single quotes).

Hence, A is printed to the output. It is because Java treats character as integral types and the ASCII value of A is 65.

String type

Java also provides support for character strings via java.lang.String class. Strings in Java are not primitive types. Instead, they are objects. For example

String myString = "Java Programming";

Here, myString is an object of the String class.




Java Operators

Java Operators

Operators are symbols that perform operations on variables and values.

Operators in Java can be classified into 5 types:

  1. Arithmetic Operators
  2. Assignment Operators
  3. Relational Operators
  4. Logical Operators
  5. Unary Operators
  6. Bitwise Operators

Java Arithmetic Operators

Arithmetic operators are used to perform arithmetic operations on variables and data. For example


a + b;

Here, the + operator is used to add two variables a and b. Similarly, there are various other arithmetic operator in Java.

Operator Operation
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulo Operation (Remainder after division)

Example : Arithmetic Operator

class HelloWorld { public static void main(String[] args) { int a = 10, b = 5; System.out.println("a+b = " + (a + b)); System.out.println("a-b = " + (a - b)); System.out.println("a*b = " + (a * b)); System.out.println("a/b = " + (a / b)); System.out.println("a%b = " + (a % b)); } }

Output


a+b = 15
a-b = 5
a*b = 50
a/b = 2
a%b = 0
